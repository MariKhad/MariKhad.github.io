Перепешите 'if' в '?'

Начальный вариант

let result;
if (a + b < 4) {
result = 'Мало';
} else {
result = 'Много'; }

Решение

let result = (a + b < 4) ? 'Мало': 'Много'

Перепешите 'if..else' в '?'

Начальный вариант

let message;
if (login == 'Сотрудник') {
message = 'Привет';
} else if (login == 'Директор') {
message = 'Здравствуйте';
} else if (login == '') {
message = 'Нет логина';
} else {
message = '';
}

Решение

let message = (login == 'Сотрудник') ? 'Привет':
(login == 'Директор') ? 'Здравствуйте':
(login == '') ? 'Нет логина':
'';
